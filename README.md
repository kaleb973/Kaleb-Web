# Kaleb-Web

Kaleb-Web is a simple personal website for Kaleb Hailu. It is a static portfolio site that highlights projects, background, and contact information.

## Live Site

The site is published on GitHub Pages:

https://kaleb973.github.io/Kaleb-Web/

## Pages

- `index.html` - Home page and entry point
- `about.html` - Background, interests, and contact information
- `projects.html` - Links to selected projects
- `404.html` - Custom not-found page

## Tech Stack

- HTML5
- CSS3
- GitHub Pages for hosting
